# TripManager
This App help people to organize trip and help user to know the best place to make tour
# ToDo App REST API
### For task management

Here is a list of all API Endpoints that you will find:

## Authentication

* POST /api/v1/signup
* POST /api/v1/login
* POST /api/v1/user
* GET /api/v1/user
* DELETE /api/v1/user/:id
* PATCH /api/v1/user/:id

## Tour API
* POST /api/v1/tour
* GET /api/v1/tour
* DELETE /api/v1/tour/:id
* PATCH /api/v1/tour/:id

## Review on tour and comment on article API
* POST /api/v1/comment
* GET /api/v1/review

## Article API
* POST /api/v1/article
* GET /api/v1/article
* GET /api/v1/article/:id
* DELETE /api/v1/article/:id
* PATCH /api/v1/article/:id

### Setup project locally

* Install [git](https://git-scm.com/downloads)
* Install [Node js](https://nodejs.org/en/)
* Install [MongoDB] or use MongoDB atlas
* Clone Repo [TripManager](https://github.com/Angelus123/TripManager)

```
$ To move into folder
```
$ cd TripManager
```
Install dependincies as they appear in package.json file by

```
$ npm install
```
To start the server do

```
$ npm run dev-start
```
To run the doc do

```

$ npm run doc
